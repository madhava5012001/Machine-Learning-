KNN ALGORITHM¶

The value of a data point is determined by the data points around itK = 1 If you have one very close friend and spend most of your time with him/her, you will end up sharing similar interests and enjoying same things. That is kNN with k=1K = 5 If you always hang out with a group of 5, each one in the group has an effect on your behavior and you will end up being the average of 5. That is kNN with k=5.kNN classifier determines the class of a data point by majority voting principle. If k is set to 5, the classes of 5 closest points are checked. Prediction is done according to the majority class

ADVANTAGE OF KNN¶

Simple and easy to interpret Does not make any assumption so it can be implemented in non-linear tasks. Works well on classification with multiple classes Works on both classification and regression tasks

DISADVANTAGE OF KNN¶

Becomes very slow as the number of data points increases because the model needs to store all data points. Not memory efficient Sensitive to outliers. Outliers also have a vote!
